# 机器人关节电机质量自动化测试模拟平台

## 项目目标
演示如何将软件质量保障中的**自动化测试、数据分析和报告生成**能力，迁移到硬件（机器人关节电机）的质量测试流程中。

## 核心价值
1.  **流程模拟**：完整模拟了从“定义测试规范”->“生成/采集数据”->“自动化判定”->“生成结构化报告”的硬件质量测试流程。
2.  **故障注入**：通过代码模拟电机过热故障，验证测试系统的有效性。
3.  **即插即用**：该项目框架可直接用于解析真实的传感器日志数据。

## 如何运行
1.  确保安装Python 3.6+。
2.  克隆本项目：`git clone https://gitee.com/你的用户名/robot-joint-motor-quality-simulator.git`
3.  运行主程序：`python main.py`

## 项目输出
程序将生成：
1.  终端显示的测试结果摘要。
2.  一个结构化的 `test_report.json` 文件，包含完整测试详情，可作为质量追溯和8D报告的基础。

## 技能体现
- **硬件测试思维**：将抽象的硬件测试规范（如温升限值）转化为具体的、可执行的代码逻辑。
- **自动化能力**：使用Python实现测试流程的完全自动化。
- **质量体系理解**：产出符合质量管理要求的标准化测试报告。
